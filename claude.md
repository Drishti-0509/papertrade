This is a MERN Stack Paper Trading Platform.

Project Structure:
- /client → React frontend (Vite)
- /server → Node.js + Express backend
- /server/models → MongoDB Mongoose models
- /server/routes → Express API routes
- /server/controllers → Business logic
- Database: MongoDB Atlas

Rules:
- Always use JWT for authentication
- Use .env for all secrets and API keys
- Use axios in frontend for API calls
- Keep frontend and backend separate
- Write clean commented code
- After each feature, test it before moving on
