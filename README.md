# PaperTrade - Algorithmic Paper Trading Platform

## Features
- User authentication with JWT
- Real-time stock prices via Alpha Vantage API
- Buy/sell stocks with $100,000 virtual balance
- Portfolio tracking with P&L
- Backtesting engine (Moving Average + RSI strategies)
- News sentiment analysis via Finnhub
- Real-time WebSocket price updates

## Tech Stack
MongoDB, Express, React, Node.js, Socket.io, Recharts

## Setup
1. Clone repo
2. cd server && npm install
3. cd client && npm install  
4. Add .env files with API keys
5. Run servers
